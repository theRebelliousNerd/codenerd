# northstar/ - Goal Alignment Atoms

Vision tracking and goal alignment guidance.

## Files

| File | Purpose |
|------|---------|
| `goal_tracking.yaml` | Session and campaign goal tracking |
| `vision_alignment.yaml` | Alignment with project vision |
| `alignment_check.yaml` | Deviation detection |
| `requirements.yaml` | Requirements tracking |
| `doc_ingestion.yaml` | Strategic document ingestion |
| `architecture_roadmap_validation.yaml` | Architecture alignment |

## Northstar System

The Northstar system maintains alignment between:
- User's stated goals
- Project vision/architecture
- Current implementation direction

## Guardian Role

The NorthstarGuardian:
- Tracks goal drift
- Validates architectural decisions
- Suggests course corrections

## Selection

Northstar atoms are selected during campaign phases and strategic planning.
