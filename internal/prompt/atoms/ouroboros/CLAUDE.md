# ouroboros/ - Self-Modification Atoms

Tool generation and self-improvement guidance.

## Files

| File | Purpose |
|------|---------|
| `tool_generator_identity.yaml` | ToolGenerator shard identity |
| `specification.yaml` | Tool specification phase |
| `refinement.yaml` | Iterative tool refinement |
| `safety_check.yaml` | Constitutional safety validation |
| `simulation_deployment.yaml` | Dream mode testing and deployment |
| `detection.yaml` | Capability gap detection |
| `adversarial_warning.yaml` | Thunderdome attack warnings |

## Ouroboros Loop

```text
Capability Gap Detected
    |
    v
Specification (define tool interface)
    |
    v
Generation (produce code + tests)
    |
    v
Safety Check (constitutional validation)
    |
    v
Thunderdome (adversarial testing)
    |
    v
Simulation (dream mode test)
    |
    v
Registration (make tool available)
```

## Selection

Ouroboros atoms are selected via `ouroboros_stage` in CompilationContext.
